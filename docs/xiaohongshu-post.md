## 正文

我终于把小爱音箱改造成了自己的网易云音乐语音入口。

以前的问题是：小爱音箱自带音乐源不好用，想听网易云歌单还要手动打开 Mac，而且“小爱同学，播放 xxx”经常被官方逻辑劫持。

所以我做了一个小项目：小爱音箱负责听我说话，Mac 负责理解我要听什么，网易云音乐客户端负责真正播放。

现在我可以直接说：

“小爱同学，智能音乐 kkecho”

“小爱同学，智能音乐 下雨天睡觉听的歌单”

“小爱同学，智能下一首”

重点是：它会优先匹配我自己的网易云歌单。比如我说“开开 echo”，也能匹配到 kkecho。我说“写代码”，会命中写代码/学习歌单。匹配不上才搜索，最后才用大模型兜底。

所以大部分时候不需要等 AI，也不烧 token。

技术链路大概是：

小爱音箱 → xiaomusic → Mac Music Agent → 网易云 macOS 客户端

还做了几个优化：

✅ 自定义唤醒词：智能音乐 / 智能播放  
✅ 本地歌单 alias 表，支持发音相似匹配  
✅ 网易云原生播放队列  
✅ 自动切随机播放  
✅ launchd 开机自启动  
✅ API key / cookie 全部本地保存，不上传  

适合：有小爱音箱、主力听网易云、家里有常开的 Mac、想折腾智能家居音乐系统的人。

项目已经整理成开源了。求一个 star，让我知道不是只有我一个人这么折腾 😂
